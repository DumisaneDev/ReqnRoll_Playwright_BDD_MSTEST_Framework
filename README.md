# ReqnRoll_Playwright_BDD_MSTEST_Framework
Enterprise BDD automation engine using .NET 10, ReqnRoll, and Playwright. Optimized for scale with 16-worker parallel execution and   SOLID architecture. Delivers forensic debugging (Screenshots, Logs, failure-only Traces) and AI-powered reporting via ReportPortal &amp;   Docker. Turning raw test data into actionable business intelligence.
